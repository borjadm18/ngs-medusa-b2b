# NGS Profile

Primer perfil cliente del template B2B.

Incluye:

- `client-profile.json`: marca, navegacion, SEO y footer.
- `homepage-content.json`: contenido base de home.
- `product-packaging.csv`: reglas B2B por SKU.

Uso previsto:

1. Importar productos y variantes NGS.
2. Importar `product-packaging.csv` desde el widget Admin o script.
3. Importar `homepage-content.json` al modulo homepage.
4. Renderizar storefront con `client-profile.json`.

Nota: hoy parte del storefront aun contiene copy NGS hardcodeado. Este perfil marca el contrato objetivo para extraerlo en la siguiente fase.
